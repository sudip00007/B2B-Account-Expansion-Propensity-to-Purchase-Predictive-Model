# B2B Account Expansion & Propensity-to-Purchase Predictive Model

## Overview

This project develops a machine learning classification pipeline to predict cross-sell and upsell readiness among corporate banking clients.

The objective is to identify high-potential accounts and enable customer success teams to proactively target expansion opportunities.

## Machine Learning Model

* Random Forest Classifier
* Class balancing
* Probability scoring
* Cross-validation
* Feature importance analysis

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC Score

## Business Impact

The model identifies high-propensity clients for:

* Cross-selling
* Upselling
* Account expansion campaigns
* Customer success prioritization

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Jupyter Notebook
